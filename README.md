TypeScript Node Skeleton
========================

TypeScript node.js project skeleton


Developing
----------


### First time install ###

    npm install


### Format, lint, build, test ###

    npm run all


### Generate documentation ###

    npm run doc


### Update dependencies ###

    npm run ncu


### Publish ###

    npm login
    npm version patch/minor/major
    npm publish
